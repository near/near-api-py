# near-api-py

*Status: super rough, APIs are subject to change*

A Python library for development of applications that are using NEAR platform.

# Contribution

First, install the package in development mode:
```bash
python setup.py develop
```

To run tests:

**Linux**
```bash
cd test
python3 -m unittest
```

**Windows**
```commandline
cd test
python -m unittest
```
